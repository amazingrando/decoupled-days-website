# Decoupled Days

## Install

Make sure you are using the right version of NPM. Note, may need to install NVM. If you get an error you may need to do an `nvm install`

```sh
nvm use
```

Install the required packages:

```sh
npm install
```

Then you can run it by:

```sh
npm run develop
```
